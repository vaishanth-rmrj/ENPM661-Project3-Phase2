Instructions to run the code
code1:
    1. python3 astar_planner.py

code2:
    1. cd catkin_ws
    2. catkin build
    3. roslaunch turtlebot3_pathplanner phase2_world.launch 
    4. cd catkin_ws/src/turtlebot3_simulations/turtlebot3_pathplanner/scripts 
        (rosrun is unable to run python 3 file so you have to cd into the scripts folder and run manually)
    5. python3 robot_handler.py
    6. Inputs:
        Robot radius = 10
        Clearance = 5
        step size = 15
